# Pfleiderer et al. 2017 Weimar Jazz Database (WJD) Solo Beat Annotations

taken from International Audio Lab Erlangen's
[JazzTube](http://mir.audiolabs.uni-erlangen.de/jazztube/static/csv_beats.zip)
website.
